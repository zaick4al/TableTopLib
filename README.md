## Custom library of custom objects for Tabletop games written on Qt/C++/STL
  This library is a part of a bigger project that is currently in early development. Uses modern C++ for smart pointers and Qt for objects, signals and slots. Currently works fine on Qt 6.5.3.
## Documentation
